Tools and Technology

Abstract

This research enhances the security of web applications by implementing a Web Application Firewall (WAF) using on ModSecurity and the Core Rules Set from OW ASP. it investigated the firewall's effectiveness in preventing attack such as Cross-Site Scripting (XSS) and SQL Injection, aiming to strengthen web applications' protection on open networks.

Methodology

This research employed an experimental method, deploying and testing the WAF on an integrated campus management system. Comparative tests were conducted on the system both before and after WAF implementation to assess the firewall's impact on attack prevention and system performance.
